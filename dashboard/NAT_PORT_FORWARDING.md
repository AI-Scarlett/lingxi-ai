# 🌐 外网访问配置指南（无需域名）

**版本：** v3.3.3  
**特点：** 纯 IP+token，零域名，零备案

---

## 🎯 方案说明

**核心思路：** 公网 IP + 端口映射 + token 认证

**无需：**
- ❌ 域名
- ❌ 备案
- ❌ 复杂配置

**需要：**
- ✅ 路由器管理员权限
- ✅ 端口映射功能
- ✅ 5 分钟配置时间

---

## 📋 配置步骤

### 步骤 1：查看本机内网 IP

**Windows：**
```bash
ipconfig
# 找到 "IPv4 地址"，例如：192.168.1.100
```

**Mac/Linux：**
```bash
ifconfig
# 找到 "inet"，例如：192.168.1.100
```

**记录这个 IP，后面要用！**

---

### 步骤 2：登录路由器后台

**常见路由器地址：**
- TP-Link：http://192.168.1.1 或 http://tplinkwifi.net
- 小米：http://192.168.31.1 或 http://miwifi.com
- 华为：http://192.168.3.1
- 华硕：http://192.168.1.1 或 http://router.asus.com
- 网件：http://192.168.1.1 或 http://routerlogin.net

**用户名/密码：**
- 默认通常在路由器背面
- 常见组合：admin/admin, admin/password, admin/1234

---

### 步骤 3：配置端口映射

**1. 找到端口映射设置**

不同路由器位置不同：
- TP-Link：应用管理 → 虚拟服务器
- 小米：高级设置 → NAT 转发 → 端口转发
- 华为：更多功能 → 网络设置 → 端口转发
- 华硕：高级设置 → 虚拟网络/端口转发
- 网件：高级 → 高级设置 → 端口转发/端口触发

**2. 添加新规则**

| 字段 | 填写内容 | 说明 |
|------|----------|------|
| 服务名称 | Lingxi | 随便填，自己认识就行 |
| 外部端口 | 8766 | 外网访问的端口 |
| 内部端口 | 8766 | Dashboard 远程访问端口 |
| 内部 IP 地址 | 192.168.1.100 | 步骤 1 查到的 IP |
| 协议 | TCP | 或"全部" |
| 状态 | 启用 | 必须启用 |

**3. 保存并应用**

点击"保存"或"应用"，等待路由器重启（约 30 秒）。

---

### 步骤 4：查看公网 IP

**方法一：百度搜索**
```
打开百度，搜索 "IP"
第一行显示的就是你的公网 IP
```

**方法二：访问网站**
```
https://www.ip138.com/
https://ifconfig.me/
```

**记录这个公网 IP！**

### 步骤 6：外网访问测试

**在外部网络（如手机 4G）：**
1. 打开手机浏览器
2. 访问：`http://公网 IP:8766?token=xxx`
3. 成功！

## 🔧 常见问题

### Q: 找不到公网 IP 怎么办？

**A:** 可能你的宽带是内网 IP，解决方法：

**方法 1：联系运营商**
```
拨打客服电话：
- 电信：10000
- 联通：10010
- 移动：10086

说："我需要公网 IP，用于监控摄像头"
（不要说用于服务器，可能不给）
```

**方法 2：使用内网穿透工具**
```
如果运营商不给公网 IP，使用：
- 花生壳（免费）
- ngrok（免费/付费）
- frp（自建，需要 VPS）
```

---

### Q: 端口映射后还是访问不了？

**A:** 检查以下几点：

**1. 防火墙设置**
```bash
# Linux 开放端口
sudo ufw allow 8766/tcp

# Windows 防火墙
控制面板 → Windows Defender 防火墙 → 高级设置
→ 入站规则 → 新建规则 → 端口 → TCP → 8766
```

**2. 路由器设置**
- 确认端口映射已启用
- 确认内部 IP 正确
- 确认端口是 8766

**3. 测试内网访问**
```bash
# 在同一局域网的其他设备访问
http://192.168.1.100:8766
```
