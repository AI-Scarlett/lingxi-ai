## 📱 手机访问（2 种方式）

### 方式一：内网访问（最快，推荐 ⭐⭐⭐⭐⭐）

**适用场景：** 手机和电脑在同一 WiFi

**步骤：**
1. 手机连接电脑同一 WiFi
2. 打开手机浏览器
3. 访问显示的"内网访问 - 手机"地址（如：http://192.168.1.100:8765/mobile）
4. 完成！

**优点：**
- ✅ 速度快
- ✅ 零配置
- ✅ 最简单

---

### 方式二：外网访问（任何地方，推荐 ⭐⭐⭐⭐⭐）

**适用场景：** 任何地方，任何网络（4G/5G/ 其他 WiFi）

**步骤：**
1. 启动 Dashboard 时会显示外网访问地址
2. 打开手机浏览器
3. 访问显示的外网地址（如：http://123.45.67.89:8766/mobile?token=xxx）
4. 完成！

**优点：**
- ✅ 任何地方都能访问
- ✅ 无需域名
- ✅ 无需备案
- ✅ Token 认证，安全

**配置：** 需要配置路由器端口映射（5 分钟）
**教程：** 查看 `NAT_PORT_FORWARDING.md`

---

### 方式三：扫码访问（最方便，推荐 ⭐⭐⭐⭐⭐）

**适用场景：** 懒得手动输入

**步骤：**
1. 启动 Dashboard 时会显示二维码
2. 手机打开微信/浏览器扫一扫
3. 自动跳转访问
4. 完成！

**优点：**
- ✅ 不用输入地址
- ✅ 很方便
- ✅ 一键访问

## 🔑 Token 管理

### 什么是 Token？

Token 就是访问密码，保护你的 Dashboard 不被别人访问。

---

### 查看 Token

```bash
python3 quick_access.py --list-tokens
```

**输出：**
```
📋 有效 Tokens (2 个):
   abc123... - user_1234567890 - 过期：2026-04-10
   def456... - user_0987654321 - 过期：2026-04-10
```

---

### 生成新 Token

```bash
python3 quick_access.py --generate-token
```

**输出：**
```
✅ 生成新 Token: abc123xyz
   外网访问：https://lingxi.me-ai.help?token=abc123xyz
```

---

### 撤销 Token

```bash
python3 quick_access.py --revoke abc123xyz
```

**输出：**
```
✅ 已撤销 Token: abc123xyz...
```

**适用场景：**
- Token 泄露了
- 不想让某人访问了
- 定期清理

## 🙋 常见问题

### Q: 我没有域名怎么办？

**A:** 不需要域名！使用官方域名 `lingxi.me-ai.help` 即可！

---

### Q: 外网访问收费吗？

**A:** 完全免费！老板提供的！

---

### Q: Token 有效期多久？

**A:** 默认 30 天，可以自定义（1 天 -365 天）。

---

### Q: 可以多人同时访问吗？

**A:** 可以！每个用户一个 token，互不影响。

---

### Q: 手机访问需要安装 APP 吗？

**A:** 不需要！手机浏览器直接访问即可。

---

### Q: 忘记了 token 怎么办？

**A:** 
```bash
python3 quick_access.py --list-tokens
```
查看所有有效 token。

---

### Q: 别人能猜到我的 token 吗？

**A:** 不能！token 是随机生成的，比密码还安全。

---

### Q: 可以自定义 token 吗？

**A:** 不可以！为了安全，必须随机生成。

---

### Q: 内网访问和外网访问有什么区别？

| 特性 | 内网访问 | 外网访问 |
|------|----------|----------|
| **速度** | 快 | 中等 |
| **范围** | 同一 WiFi | 任何地方 |
| **配置** | 零配置 | 零配置 |
| **安全性** | 中 | 高（HTTPS） |
| **推荐** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
