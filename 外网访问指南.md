## ⚠️ 注意事项

1. **防火墙**: 确保服务器 8765 端口已开放
   ```bash
   # 检查防火墙状态
   ufw status
   
   # 开放端口（如果需要）
   ufw allow 8765/tcp
   ```

2. **安全组**: 如果在云服务器上，需要在云控制台开放 8765 端口

3. **Token 认证**: 当前未启用强制认证，生产环境建议启用

4. **HTTPS**: 当前使用 HTTP，生产环境建议配置 Nginx + HTTPS
